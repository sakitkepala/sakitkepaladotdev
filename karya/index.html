sumberkunjungi

Web site tool simpel yang secara kegunaan lumayan membantu untuk buat dan generate resume selembar halaman dalam bentuk file PDF.

Karena ini projek sampingan, dan saya juga mau coba-coba bikin sesuatu yang "fancy", UI "form" untuk input data resume saya buat supaya mirip editor markdown. Jadi bukan benar-benar markdown, tapi developer pasti bisa langsung mengenali maksudnya kalau dicoba 😁

Note App yang Terinspirasi oleh Zettelkasten

sumberkunjungi

Proof of concept untuk aplikasi penulisan catatan yang mencoba mengadopsi konsep Zettelkasten. Sekaligus dibuat sebagai projek model untuk meriset beberapa teknologi sekaligus, seperti Nx Workspace, GraphQL API dan GraphQL client, Fastify, Vanilla Extract CSS, React Native, Framer Motion, dan lainnya.

Aplikasinya belum ready, dan saya masih coba-coba eksplorasi konsep UI-nya juga. Tapi kode di repositori bisa di-clone untuk jalan di localhost—barangkali penasaran.
